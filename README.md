# AI Driven Legal and Risk Intelligence Platform

An end to end data pipeline and machine learning system for extracting insights from large scale legal and regulatory documents.

## Features

- Automated ingestion of legal cases using CourtListener API
- NLP based entity extraction from unstructured legal text
- Machine learning models for legal risk classification
- Interactive analytics dashboards

## Architecture

Legal Data API → ETL Pipeline → NLP Processing → ML Risk Models → Analytics Dashboard

NLP (spaCy)
